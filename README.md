# Storybook for Tamagui with Expo

Storybook for [tamagui](https://github.com/tamagui/tamagui) with Expo

## Get started

### Web

```sh
bun storybook:web
```

### Mobile

> [!WARNING]
> I haven't tested iOS so I can't guarantee it works

```sh
# both ios and android
bun storybook
```

```sh
# android
bun storybook:android
```

```sh
# ios
bun storybook:ios
```
